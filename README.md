# Dashboard Comercial Purion

Dashboard estático para GitHub Pages que lê a planilha `crm_purion.xlsx` e recalcula automaticamente os indicadores comerciais.

## Como atualizar

1. Atualize a planilha CRM no Excel mantendo a aba `CRM_Leads`.
2. Substitua o arquivo `crm_purion.xlsx` neste repositório, mantendo exatamente o mesmo nome.
3. Aguarde alguns instantes e recarregue a página publicada no GitHub Pages.

## Estrutura

- `index.html` — dashboard dinâmico.
- `crm_purion.xlsx` — base de dados do CRM.
- `README.md` — instruções.

## Página publicada

Quando o GitHub Pages estiver ativo, a página ficará em:

https://fagnermelo.github.io/dashboardpurion/
